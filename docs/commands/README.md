# Commands by Platform

Commands are split by MCP platform. Set `PLATFORM=browser` or `PLATFORM=node` in your MCP config.

## Browser Platform (`PLATFORM=browser`)

See [browser-reference](./browser-reference.md) for web automation commands: browse, screenshot, click, fill, accessibility, console, network, etc.

## Node Platform (`PLATFORM=node`)

See [node-reference](./node-reference.md) for Node.js backend debugging: connect, tracepoint, logpoint, snapshots, run-js-node.
